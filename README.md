A simple to-do-list App for exercise purposes.

Built with React/ HTML5/ CSS3.

Here we try to create an array of tasks and use the useState hook to add new tasks to it.
We populate an ordered list with the map method, so that each list item contains a task from the tasks array.
We also add the feature of moving tasks up or down the list, or deleting them.
Some CSS styling is added for polishing of our UI.
